Coffee Lover Bootstrap Project

Welcome to Bootstrap Project : Coffee Lovers Blog

We have to recreate this webpage and additional form page

Follow the instructions given below to recreate the webpage

  1. Create a general Landing page with a jumbotron
  2. Next create a NavBar on the page that links to another html file
  3. Add two paragraphs of lorem ipsum below the Jumbotron
  4. Next use the Bootstrap Grid system to add in thumbnails of coffee : you will need to use the thumbnail class
    • One
    • Two

  5. Next make sure that at the largest setting you have 3 columns of pictures, on the smallest setting you should have 2 columns. Its up to us where we want to change
  6. Next create another HTML file for the Sign-Up page
  7. Crete a form o the Sign-Up page with e-mail, password and a check-box

That's it, feel free to play around with this project and style it more to your liking. The main focus of this project are the uses of Navbar, Jumbotron, Form, Container and Grid System
